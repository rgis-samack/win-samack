## 🇧🇷 Português

O **Samack WinUtil** é uma suíte completa e portátil de otimização, debloat e gerenciamento do sistema Windows. Desenvolvido nativamente em **PowerShell** com uma interface moderna e responsiva construída em **WPF (Windows Presentation Foundation)**, o utilitário permite que você faça limpezas profundas, desinstalações limpas de softwares, diagnósticos de rede e ativações seguras de forma prática e rápida.

---

### 🚀 Como Executar Online (Sem Baixar Arquivos)

Abra o seu PowerShell como **Administrador** e execute o seguinte comando:

```powershell
irm https://raw.githubusercontent.com/rgis-samack/win-samack/refs/heads/main/win-samack.ps1 | iex
```

Ou através do link encurtado oficial:

```powershell
irm https://bit.ly/win-samack | iex
```

> 💡 **Nota de Compatibilidade:** O carregador em código ASCII puro contorna erros de codificação de caracteres em qualquer versão do Windows, puxando o script em codificação UTF-8 com BOM direto da memória.

---

### 📋 Funcionalidades Detalhadas

* **💻 Painel Principal:** Monitoramento de hardware em tempo real (CPU, RAM, Uptime), limpeza rápida de memória RAM e processos de alto consumo.
* **🧹 Limpeza de Disco:** Remoção profunda de arquivos temporários, cache do Windows Update, logs do sistema e arquivos inúteis.
* **🧹 Debloat (Apps):** Remova aplicativos integrados indesejados (como Cortana, Bing, Telemetria) para liberar recursos e privacidade.
* **🎮 Desempenho & Jogos:** Otimizações avançadas no registro do Windows para reduzir input lag, otimizar consumo de energia e serviços desnecessários.
* **📥 Instalar Apps:** Instalação silenciosa e automatizada em lote dos aplicativos mais comuns do mercado (via WinGet).
* **🗑️ Desinstalar Apps:** Varredura profunda e completa no registro do sistema para desinstalações limpas e remoção de pastas órfãs.
* **🌐 Ferramentas de Rede:** Diagnóstico com IPConfig, Ping e Tracert fluidos (sem travar a tela), Netstat, Flush DNS e Reset de Rede.
* **🔑 Ativação (MAS):** Integração segura e de código aberto (MAS) para ativação digital permanente do Windows e Office.
* **📦 Downloads do Office:** Nova aba dedicada com instaladores e downloads oficiais das edições do **Microsoft Office 2019** e **Office 2021** através de links encurtados rápidos.
* **📜 Logs de Execução:** Histórico transparente de todas as ações realizadas no sistema.

### 📋 Detailed Features

* **💻 Main Dashboard:** Real-time hardware monitoring (CPU, RAM, Uptime), quick RAM cleanup, and high-resource process display.
* **🧹 Disk Cleanup:** Deep removal of temporary files, Windows Update cache, system logs, and garbage files.
* **🧹 Debloat (Apps):** Remove unwanted pre-installed Windows apps (Cortana, Bing, Telemetry) to free resources and protect privacy.
* **🎮 Performance & Gaming:** Advanced Windows registry optimizations to reduce input lag, optimize power consumption, and disable unnecessary services.
* **📥 Install Apps:** Silent and automated batch installation of the most common applications (via WinGet).
* **🗑️ Uninstall Apps:** Deep registry scan for clean software uninstallation and residual folder cleanup.
* **🌐 Network Tools:** Network diagnostics with smooth (non-freezing) IPConfig, Ping, Tracert, Netstat, Flush DNS, and Network Reset.
* **🔑 Activation (MAS):** Secure, open-source MAS integration for permanent Windows and Office activation.
* **📦 Office Downloads:** Dedicated tab containing official downloads for **Microsoft Office 2019** and **Office 2021** via quick shortlinks.
* **📜 Execution Logs:** A transparent history of all system operations.
